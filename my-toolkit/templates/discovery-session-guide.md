## Agenda (60 min)

| Tiempo | Actividad |
|--------|-----------|

| 5-25 min | Bloque 1: Proceso actual (como funciona hoy) |
| 25-40 min | Bloque 2: Fricciones y oportunidades (donde duele) |
| 40-50 min | Bloque 3: Vision (como les gustaria que funcionara) |
| 50-60 min | Resumen + proximos pasos |

## Preguntas por Dimension

### Estrategia
- Como definen la estrategia de una campana hoy?
- Que informacion usan del brief? Que suele faltar?
- Tienen framework de propuesta o se empieza de cero?

### Operacion
- Como coordinan tareas entre el equipo?
- Donde se pierde informacion entre personas?
- Que herramientas usan para tracking?

### Contenido
- Como generan insights para reportes?
- Cuanto tiempo tarda armar un reporte final?
- Reutilizan informacion de campanas pasadas?

### Reporting
- De donde sacan la data para reportes?
- Tienen templates o cada reporte es diferente?
- Que metricas les piden los clientes?

### Cultura
- Han usado herramientas de IA antes? Cuales?
- Que esperan de los agentes de IA?
- Que les preocupa?
