## Documentos en Google Drive

La mayoría de briefs y propuestas de clientes están en Drive, NO en el workspace local. Para accederlos:

```bash
# Buscar archivos en Drive
gog drive search "brief [marca]" --max 10

# Descargar PDF de Drive para leerlo
mkdir -p /tmp/briefs
gog drive download <fileId> --output /tmp/briefs/nombre.pdf
pdf /tmp/briefs/nombre.pdf

# Leer Google Doc directamente
gog docs cat <docId>
```
