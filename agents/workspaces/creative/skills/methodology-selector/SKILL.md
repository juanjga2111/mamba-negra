# Methodology Selector — RAYO / ARCO / PRISMA / MAREA

Este skill guia la seleccion del modelo de campana correcto segun el alcance, tiempo, objetivos y complejidad del proyecto. Paso 08 del flujo estrategico.

**Prerequisito**: Necesitas brief + retos + concepto (idealmente). Si solo tienes el brief, puedes recomendar metodologia inicial, pero sera mas precisa con concepto definido.

---

## LAS 4 METODOLOGIAS DE MAMBA NEGRA

### RAYO — Campana Express

Campana corta y agil. Lineamientos directos, guiones completos desde el inicio para facilitar grabacion inmediata.

| Aspecto | Detalle |
|---------|---------|
| Timeline | 1-2 semanas |
| Complejidad | Baja |
| Influencers | Pocos, directos |
| Fases | Una sola oleada |
| Contenido | Guiones completos desde el dia 1 |

**Cuando usar RAYO**:
- Timeline apretado (el cliente necesita resultados ya)
- Brief claro y acotado — pocas variables
- Producto o servicio puntual — no requiere arco narrativo
- Presupuesto limitado de influencers

**Cuando NO usar RAYO**:
- Si hay multiples objetivos o submarcas
- Si se necesita construir percepcion a lo largo del tiempo
- Si el concepto tiene potencial de desarrollo por fases

---

### ARCO — Campana por Fases

Su diferencial no es la duracion, sino la existencia de **momentos distintos** que estructuran la narrativa.

| Aspecto | Detalle |
|---------|---------|
| Timeline | 3-6 semanas |
| Complejidad | Media |
| Influencers | Grupos por fase |
| Fases | 2-4 momentos con identidad propia |
| Contenido | Propuesta unificada que evoluciona |

**Cuando usar ARCO**:
- El concepto tiene potencial de desarrollarse en capitulos
- Se quiere construir expectativa → revelacion → cierre
- Hay historia que contar, no solo producto que mostrar
- El cliente quiere una narrativa, no solo cobertura

**Cuando NO usar ARCO**:
- Si no hay arco narrativo natural en el concepto
- Si el timeline es demasiado corto para fases

---

### PRISMA — Campana Robusta

Alto volumen, piramide de distribucion, precision en tiers y etapas del funnel.

| Aspecto | Detalle |
|---------|---------|
| Timeline | 4-8 semanas |
| Complejidad | Alta |
| Influencers | Piramide por tier y funnel |
| Fases | Multiples, con objetivos distintos por etapa |
| Contenido | Diferenciado por tier, fase y objetivo |

**Cuando usar PRISMA**:
- Multiples objetivos (awareness + consideracion + conversion)
- El cliente tiene submarcas o lineas de producto
- Se necesita precision en la seleccion de influencers por etapa
- Presupuesto amplio que permite escala
- El concepto se puede ejecutar de distintas formas segun el tier

**Cuando NO usar PRISMA**:
- Si el brief es simple y acotado
- Si el presupuesto no permite escala
- Si el producto es uno solo y directo

---

### MAREA — Always On

Presencia sostenida, embajadores a largo plazo, contenido que fluye y se adapta por ciclo.

| Aspecto | Detalle |
|---------|---------|
| Timeline | 6-12 meses |
| Complejidad | Continua |
| Influencers | Embajadores de largo plazo |
| Fases | Ciclos (mensuales, bimestrales) con focos |
| Contenido | Propuesta unificada que se adapta por ciclo |

**Cuando usar MAREA**:
- La marca necesita presencia constante, no picos puntuales
- Hay presupuesto para sostener la relacion con influencers a largo plazo
- El objetivo es brand building y comunidad, no activacion puntual
- Se quiere construir embajadores autenticos de la marca

**Cuando NO usar MAREA**:
- Si es una campana puntual con fecha de cierre
- Si el presupuesto no soporta 6+ meses
- Si el concepto funciona mejor como impacto unico

## PROCESO DE SELECCION

### Paso 1: Evaluar los inputs

Necesitas:
1. **Timeline del brief** — cuanto tiempo hay
2. **Objetivo principal** — awareness, conversion, brand building, lanzamiento
3. **Numero estimado de influencers** — pocos vs muchos
4. **Complejidad del concepto** — lineal vs multi-fase
5. **Presupuesto** — acotado vs amplio

### Paso 2: Recomendar con justificacion

```
METODOLOGIA RECOMENDADA: [RAYO / ARCO / PRISMA / MAREA]

POR QUE:
1. [razon basada en el brief — timeline, objetivo, complejidad]
2. [razon basada en el concepto — como se despliega mejor con esta metodologia]
3. [razon basada en los influencers — cantidad, tier, dinamica]

ALTERNATIVA VIABLE: [otra metodologia que podria funcionar]
- Pro: [que ganarias]
- Contra: [que perderia]

FASES SUGERIDAS (si aplica):
Fase 1: [nombre] — Semana X-Y — Objetivo: [que busca]
Fase 2: [nombre] — Semana X-Y — Objetivo: [que busca]
[etc.]
```

### Paso 3: Validar con el equipo

**Pregunta a Mar**: "Te hace sentido esta metodologia o ves otra opcion?"

Si hay duda entre dos metodologias, presenta ambas con pro/contra y deja que Mar decida.

---

## REGLAS

1. **La metodologia sirve al concepto** — no al reves. No fuerces un concepto en una metodologia que no le queda.
2. **Siempre justifica** — no digas "ARCO" sin explicar por que. El equipo necesita entender la logica.
3. **Presenta alternativa** — siempre menciona una segunda opcion viable con su trade-off.
4. **Lee campaign-framework.md** — `knowledge/campaign-framework.md` seccion 03 tiene el detalle completo. Consultalo via `memory_search "tipos de campana"`.
5. **Mar decide** — tu recomiendas con conviccion, ella elige.
6. **Desarrolla a fondo** — la justificacion debe ser solida. Minimo 3-5 lineas por cada razon.
